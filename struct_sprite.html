Detailed Description

Standard DynamicSceneNode inheritor to render some texture. Texture can be animated and can have a lot of animations. Animations stores in plane texture like this: A1Frame1 A1Frame2 A1Frame3 ... A1FrameN A2Frame1 A2Frame2 ... So, frames of animation places in texture horizontally. Animations itself places in texture vertically. Frame size of each animation is constant. Animations can have defferent count of frames and theirs duration.

Warning
Sprite is an inheritor of the DynamicSceneNode. It means, that other code is use pointer casting to Sprite from DynamicSceneNode and vise versa. You SHOULD include the struct DynamicSceneNode blablaNode; at the begining of Sprite struct, if you want code to work with Sprite like with a DynamicSceneNode. More, you SHOULD initialize blablaNode by calling SceneNode_initDynamic(). Also function pointers in DynamicSceneNode::sceneNode SHOULD be initialized to your function implementation. Don't forget to add this warning comment to your own new DynamicSceneNode inheritors.
