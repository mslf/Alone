Constructs Scene and inits it from TextParser.

Parameters
resourceManagerPointer to a ResourceManager for loading all needed resources. Can be NULL.
rendererPointer to a Renderer for creating SceneNode, that needs Renderer. Can be NULL.
sceneNodeTypesRegistrarPointer to a SceneNodeTypesRegistrar for creating any registered beforeSceneNode. Can be NULL.
sceneResIdString with ID (path) to the Scene resource. Can be NULL.
Returns
Pointer to a Scene or NULL if something general failed.
Note
Scene will be created even if some SceneNode or ScriptResource creating failed. So returning empty Scene is normal.
Use this function in pair with Scene_destruct().
