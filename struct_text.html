Detailed Description

Standart DynamicSceneNode inheritor to render some (typically static) text.

Warning
Text is an inheritor of the DynamicSceneNode. It means, that other code is use pointer casting to Text from DynamicSceneNode and vise versa. You SHOULD include the struct DynamicSceneNode blablaNode; at the begining of Text struct, if you want code to work with Text like with a DynamicSceneNode. More, you SHOULD initialize blablaNode by calling SceneNode_initDynamic(). Also function pointers in DynamicSceneNode::sceneNode SHOULD be initialized to your function implementation. Don't forget to add this warning comment to your own new DynamicSceneNode inheritors.
