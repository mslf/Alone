struct  ContextMenu
 Standart Gui element for drawing menu, builded from Button. All menuOptions are have the similar to theirs prototypes design, but events and labels can be unique for each menuOption. User can click the ContextMenu, and some things will happen (such a GameEvent or sound playing in appropriate Button). More...

const char *const  labelText 
)

Adds new menuOption to the ContextMenu::menuOptionsList with specified data. Also, it increases ContextMenu::menuOptionsCount and changes ContextMenu::allocatedMenuOptions, if needed.

Parameters
contextMenuPointer to a #ContetMenu, where to add new menuOption. Can be NULL.
resourceManagerPointer to a ResourceManager which is used to load Button resoures for constructing new menuOption. Can be NULL.
rendererPointer to a Renderer which is used to construct Button for constructing new menuOption. Can be NULL.
sceneNodeTypesRegistrarPointer to a SceneNodeTypesRegistrar which is used to construct Button for constructing new menuOption. Can be NULL.
focusedEventResString with ID of new GameEvent, which will be set to new menuOption Button::focusedEventResource and Button::focusedGameEvent.
pressedEventResString with ID of new GameEvent, which will be set to new menuOption Button::pressedEventResource and Button::pressedGameEvent.
labelTextString to set new menuOption Button::label::text. Should be unique in the ContextMenu::menuOptionsList.
Returns
ContextMenuSceneNode_errors value.
Note
This function causes CotextMenu_updateMenuOptionsList().
