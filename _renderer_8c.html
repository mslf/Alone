)

Constructs Renderer, inits it and SDL_image and SDL_ttf.

Parameters
loggerPointer to a Logger to set Renderer::logger. Can be NULL.
settingsPointer to a Settings, where Renderer will grab display and window settings. Can be NULL.
Returns
Pointer to an initialized Renderer or NULL if something failed (mempry allocating or initializing SDL subsystems).
Note
Use this function in pair with Renderer_destruct().
