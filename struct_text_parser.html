Detailed Description

Represents structure for containing Pair, parsed from TextResource. Some info about parsing format:

Note
TextParser parses TextResource::text for assigments like this: lvalue = rvalue;, and this: lvalue = [item1, item2, item3, ..., itemN]; Also, you can quote your rvalue in quotes '"' for protecting spaces: lvalue1 = "rvalue 1"; lvalue2 = ["item 1", "item 2"]; You can add a multiline comments like in C in any place of TextResource::text (remove'_' symbol). /_* comment *_/
