Detailed Description

Standard PhysicalSceneNode inheritor which is Body with some number of SoundResource, input-triggered GameEvent, InventoryItem and MicroSection.

Warning
StandardUser is an inheritor of the PhysicalSceneNode. It means, that other code is use pointer casting to StandardUser from PhysicalSceneNode and vise versa. You SHOULD include the struct PhysicalSceneNode blablaNode; at the begining of StandardUser struct, if you want code to work with StandardUser like with a PhysicalSceneNode. More, you SHOULD initialize blablaNode by calling SceneNode_initPhysical(). Also function pointers in PhysicalSceneNode::sceneNode SHOULD be initialized to your function implementation. Don't forget to add this warning comment to your own new PhysicalSceneNode inheritors.
