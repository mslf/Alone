struct  Sprite
 Standart DynamicSceneNode inheritor to render some texture. Texture can be animated and can have a lot of animations. Animations stores in plane texture like this: A1Frame1 A1Frame2 A1Frame3 ... A1FrameN A2Frame1 A2Frame2 ... So, frames of animation places in texture horizontally. Animations itself places in texture vertically. Frame size of each animation is constant. Animations can have defferent count of frames and theirs duration. More...

enum SpriteSceneNode_errors

Error codes for Sprite.

Enumerator
SPRITE_NO_ERRORS 

All right, no errors.

SPRITE_ERR_NULL_ARGUMENT 

Some of function's arguments are NULL.

SPRITE_ERR_GETTING_TEXTURE_SIZE 

Getting size of Sprite::textureResource failed.

SPRITE_ERR_FRAME_SIZE_NOT_FIT_H 

GIven (or divided by 2) frame width multiplyed by framesCount doesn't fit into Sprite::textureResource.

SPRITE_ERR_FRAME_SIZE_NOT_FIT_V 

GIven (or divided by 2) frame height multiplyed by animationsCount doesn't fit into Sprite::textureResource.

SPRITE_ERR_ANIMATIONS_ALLOC 

Allocating memory for Sprite::animations failed.

SPRITE_ERR_CONSTRUCTING_TEXTURE_RES 

Constructing Sprite::textureResource failed.

SPRITE_ERR_CONSTRUCTIG_TEXT_PARSER 

Constructing new TextParser for some required reason failed.

SPRITE_ERR_SAVING 

Saving Sprite failed due to internal reason.

Changes Sprite::textureResource. Animations structure won't be changed. New texture will be checked to be suitable for existing Sprite::animations and Sprite::frameSize.

Parameters

resourceManagerPointer to a ResourceManager which will be used to load TextureResource. Can be NULL.
rendererPointer to a Renderer which will be used for constructing TextureResource. Can be NULL.
textureResIdString with ID (path) of the TextureResource. Can be NULL.
