§ EventManager_generateCustomEventsList()

enum EventManager_errors EventManager_generateCustomEventsList ( struct EventManager *  em,
const char *const  channel 
)

Adds GameEvent to a EventManager::customGameEventsList if GameEvent::channel == channel. Also adds GameEvent if it is broadcasting. Adds GameEvents from EventManager::gameEventsList. Also, increases EventManager::customGameEventsCount and tryies to reallocate memory if needed.

Parameters
emPointer to a EventManager. Can be NULL.
channelPointer to a const channel string. Can be NULL. GameEvents with the same channel string and with NULL channel (broadcasting) will be added to EventManager::customGameEventsList.
Returns
EventManager_errors value.
See also
GameEvent
GameEvent_construct()
EventManager_errors
Note
Last changes in EventManager::customGameEventsList will be lost!
Function firstly reset EventManager::customGameEventsCount to a 0.
