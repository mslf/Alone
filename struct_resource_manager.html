Detailed Description

Resource subsystem. Loads and stores various resources in arrays. Provide effective managment of same resources (each resources will be stored in array once). There is only one exeption: unique TextResource. If loader of TextResource wants, he can load one TextResource twice and it will be stored in array twice.

Note
ResourceManager takes care of each resource (with only one exeption for ResourceManager_loadTextureResourceFromText()), which was loaded by ResourceManager_loadTextureResource(), ResourceManager_loadTextureResourceFromText(), ResourceManager_loadTextResource(), ResourceManager_loadScriptResource() and ResourceManager_loadSoundResource(). So, only ResourceManager should destruct these resources.
