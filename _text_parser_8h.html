§ TextParser_errors

enum TextParser_errors

Error codes for TextResource.

Enumerator
TEXT_PARSER_NO_ERRORS 

All right, no errors.

TEXT_PARSER_ERR_NULL_ARGUMENT 

Some of function's arguments are NULL.

TEXT_PARSER_ERR_NO_LEFT_VALUE_IN_LIST 

No such Pair::leftValue found in any Pair in TextParser::pairsList.

TEXT_PARSER_ERR_CONVERTING 

Converting string to some value failed.

TEXT_PARSER_ERR_OUT_OF_RANGE 

RightValue::itemsList have only RightValue::itemsCount, but you are trying to get more.

TEXT_PARSER_ERR_ALLOC_STRING 

Allocating memory for some string while parsing or adding or converting failed.

TEXT_PARSER_ERR_REALLOC_PAIRS_LIST 

Allocating memory for TextParser::pairsList failed.

TEXT_PARSER_ERR_REALLOC_ITEMS_LIST 

Allocating memory for RightValue::itemsList failed.

TEXT_PARSER_ERR_SPLITTING_ASSIGMENT 

Splitting assigment string failed due to syntax error.

TEXT_PARSER_ERR_SPLITTING_ASSIGMENT_EOF 

Splitting assigment string failed due to unexpected EOF.

TEXT_PARSER_ERR_DELETING_NOT_QUOTED_SPACES 

Deleting space-symbols from right value string failed due to odd count of quotes or unexpected EOF.

TEXT_PARSER_ERR_PARSING_ITEMS_ARRAY_EOF 

Parsing items array string failed due to unexpected EOF.

TEXT_PARSER_ERR_PARSING_ITEMS_ARRAY_BRACKETS 

Parsing items array string failed because opening or closing square brackets haven't found.

TEXT_PARSER_ERR_DELETING_COMMENTS 

Syntax error or unexpected EOF happened while deleting comment blocks.

§ TextParser_standartTypeString

const char* const TextParser_standartTypeString = "type"
static

Standart type string for all typed resources, which are loaded from TextParser. TextParser_getString(TextParser, TextParser_standartTypeString, 0) should return string, which is represent type of TextResource, which was parsed by TextParser.
