Constructs new SceneNode inheritor by assotiated constructor.

Parameters
resourceManagerPointer to a ResourceManager to loading resources, which needed for SceneNode inheritor. Can be NULL.
rendererPointer to a Renderer to construct textures, which needed for SceneNode inheritor. Can be NULL.
sceneNodeTypesRegistrarPointer to a SceneNodeTypesRegistrar with registered types. Can be NULL.
resIdString with ID (path) to the SceneNode inheritor resource. Can be NULL.
requiredTypeStringString with wishful SceneNode inheritor type. Can be NULL. If it is NULL, then any registered SceneNode type will be constructed (if its resource contains apropriate type string).
Returns
Pointer to a SceneNode inheritor type object casted to a SceneNode, or NULL if failed.
