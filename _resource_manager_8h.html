struct  ResourceManager
 Resource subsystem. Loads and stores various resources in arrays. Provide effective managment of same resources (each resources will be stored in array once). There is only one exeption: unique TextResource. If loader of TextResource wants, he can load one TextResource twice and it will be stored in array twice. More...

§ ResourceManager_loadTextResource()

struct TextResource* ResourceManager_loadTextResource ( struct ResourceManager *  rm,
const char *const  textResId,
bool  unique 
)

Loads TextResource from filesystem and adds it to the ResourceManager::textResourcesList or finds already exsisting the same TextResource in ResourceManager::textResourcesList (and increases its TextResource::pointersCount). Also, ResourceManager::textResourcesCount will be increased and ResourceManager::allocatedTextResourcesCount (if needed), if new TextResource loaded.

Parameters
rmPointer to a ResourceManager where to store TextResource. Can be NULL.
textResIdString with a path to the TextResource file in filesystem. Can be NULL.
uniqueFlag to load TextResource twice and more from filesystem, even it exists in ResourceManager::textResourcesList.
Returns
Pointer to a TextResource from ResourceManager::textResourcesList, or NULL if something failed.
See also
TextResource
Note
Please, don't try to destruct returned TextResource.
This function will find non-unique TextResource in ResourceManager::textResourcesList only if you've passed false unique flag. If unique flag is true or TextResource is unique, then nothing will be found. So, you can't access unique TextResource from outside world, if someone who loaded this TextResource is not you (or not in your scope right now).

§ ResourceManager_loadTextureResourceFromText()

struct TextureResource* ResourceManager_loadTextureResourceFromText ( struct ResourceManager *  rm,
struct Renderer *  renderer,
const char *const  text,
const char *const  fontPath,
int  size,
SDL_Color  color 
)

Generates TextureResource from text and adds it to the ResourceManager::textureResourcesList or finds already exsisting the same TextureResource in ResourceManager::textureResourcesList (and increases its TextureResource::pointersCount). Also, ResourceManager::textureResourcesCount will be increased and ResourceManager::allocatedTextureResourcesCount (if needed), if new TextureResource loaded.

Parameters
rmPointer to a ResourceManager where to store TextureResource. Can be NULL. If you pass NULL here, then function will generates TextureResource without saving it in ResourceManager::textureResourcesList! So, you should destruct it by yourself.
rendererPointer to a Renderer for constructing TextureResource. Can be NULL.
textString to be rendered in the TextureResource. Can be NULL.
fontPathString with a path to the ttf font in the filesystem. Can be NULL.
sizeVertical size of text in TextureResource in Upixels.
colorSDL_Color of text in TextureResource.
Returns
Pointer to a TextureResource from ResourceManager::textureResourcesList, or NULL if something failed.
See also
TextureResource
Note
Please, don't try to destruct returned TextureResource (with exeption).
Warning
Passing NULL ResourceManager changes behaviour of the function. All returned TextureResource, generated with NULL ResourceManager, should be destructed by you.
