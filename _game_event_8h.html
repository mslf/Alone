struct GameEvent* GameEvent_construct ( const char *const  channel,
struct SceneNode *  sender,
const char *const  command,
const char *const  data 
)

Constructs GameEvent and inints it.

Parameters
channelString to set GameEvent::eventChannel string. Can be NULL. If channel is NULL, then GameEvent will be global. May be useful for broadcasting events.
senderPointer to a SceneNode which sends that GameEvent. Can be NULL. If sender is NULL, then we think, that there is no specific sender, or it's jsut not a SceneNode.
commandString to set GameEvent::command string. Can be NULL. If command is NULL, then we think that GameEvent is just a notification.
dataString to set GameEvent::data string. Can be NULL. If data is NULL, then we think that GameEvent don't need any data. May be useful for commands without arguments.
Returns
Pointer to an initialized GameEvent, or NULL if constructing failed.
Note
Only one (command or data) string can be NULL at one time.
