Adds new menuOption to the ContextMenu::menuOptionsList with specified data. Also, it increases ContextMenu::menuOptionsCount and changes ContextMenu::allocatedMenuOptions, if needed.

Parameters
contextMenuPointer to a #ContetMenu, where to add new menuOption. Can be NULL.
resourceManagerPointer to a ResourceManager which is used to load Button resoures for constructing new menuOption. Can be NULL.
rendererPointer to a Renderer which is used to construct Button for constructing new menuOption. Can be NULL.
sceneNodeTypesRegistrarPointer to a SceneNodeTypesRegistrar which is used to construct Button for constructing new menuOption. Can be NULL.
focusedEventResString with ID of new GameEvent, which will be set to new menuOption Button::focusedEventResource and Button::focusedGameEvent.
pressedEventResString with ID of new GameEvent, which will be set to new menuOption Button::pressedEventResource and Button::pressedGameEvent.
labelTextString to set new menuOption Button::label::text. Should be unique in the ContextMenu::menuOptionsList.
Returns
ContextMenuSceneNode_errors value.
Note
This function causes CotextMenu_updateMenuOptionsList().

§ CotextMenu_updateMenuOptionsList()

static enum ContextMenuSceneNode_errors CotextMenu_updateMenuOptionsList ( struct ContextMenu *  contextMenu,
struct ResourceManager *const  resourceManager,
struct Renderer *const  renderer,
struct SceneNodeTypesRegistrar *  sceneNodeTypesRegistrar 
)
static

Updates ContextMenu::optionsList in some way. Update method depends on ContextMenu::menuOptionsCount, so here is example: O - onlyOneMenuOptionPrototype, T - topMenuOptionPrototype, M - middleMenuOptionPrototype, L - lowerMenuOptionPrototype. So, if ContextMenu::menuOptionsCount == 1 then ContextMenu will looks like this [0. < O >]. If ContextMenu::menuOptionsCount == 2 then ContextMenu will looks like this [0. < T >], [1. < L >]. If ContextMenu::menuOptionsCount == 3 then ContextMenu will looks like this [0. < T >], [1. < M >], [2. < L >]. And so on. If some error happened than < O > prototype will be used.

Parameters
contextMenuPointer to a ContextMenu, where ContextMenu::optionsList will be updated.
resourceManagerPointer to a ResourceManager for loading needed by ContextMenu_updateMenuOption() resources.
rendererPointer to a Renderer for constructing Button by ContextMenu_updateMenuOption().
sceneNodeTypesRegistrarPointer to a SceneNodeTypesRegistrar for constructing Button by ContextMenu_updateMenuOption().
Returns
ContextMenuSceneNode_errors value.
