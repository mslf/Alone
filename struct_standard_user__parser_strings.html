Detailed Description

Strings which are used for constructing StandardUser from TextParser.

Note
StandardUser_parserStrings::iventoryItemsList doesn't contain actual settings (such an ID or count) of the constructed InventoryItem. It contains array of InventoryItem names, which will be found in TextParser, so they will contain actual data.
StandardUser_parserStrings::eventsList doesn't contain actual settings (such an ID or trigger scancode string) of the constructed TriggeredGameEvent. It contains array of TriggeredGameEvent names, which will be found in TextParser, so they will contain actual data.
StandardUser_parserStrings::nanoSections doesn't contain actual settings (such an ID or valueString) of the constructed NanoSection. It contains array of NanoSection names, which will be found in TextParser, so they will contain actual data.
