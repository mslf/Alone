Detailed Description

Standart Gui element for list box. Normally draws ListBox::textBox and ListBox::button. If user clicks ListBox::button, then ListBox::contextMenu will be displayed. User can choose some menu option, and it will appears in ListBox::textBox.

Warning
ListBox is an inheritor of the SceneNode. It means, that other code is use pointer casting to ListBox from SceneNode and vise versa. You SHOULD include the struct SceneNode blablaNode; at the begining of ListBox struct, if you want code to work with ListBox like with a SceneNode. More, you SHOULD initialize function pointers in blablaNode to NULL (by calling SceneNode_init()) or to your function implementation. Don't forget to add this warning comment to your own new SceneNode inheritors.
